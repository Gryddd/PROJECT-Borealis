About This Project

Borealis is a comprehensive MERN stack e-commerce project designed to demonstrate a wide range of web development skills.

It functions as a complete platform, featuring user authentication, a product catalog, guest and user shopping carts, a full checkout process with Stripe integration, and a complete admin dashboard for managing products, orders, and users. Below is a detailed look at the technology stack.
